\doxysubsection{Detailed Description}
\doxylink{classQWK_1_1StyleAgent}{Style\+Agent} provides some features related to system theme. 

Qt6.\+6 started to support system theme detection, this class is intended as an auxiliary support for lower versions of Qt. If your Qt already supports it, it is recommended that you don\textquotesingle{}t include this class in your build system.
