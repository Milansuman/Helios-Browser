\doxysubsection{Detailed Description}
\doxylink{classQWK_1_1WindowAgentBase}{Window\+Agent\+Base} is the base class of the specifiy window agent for Qt\+Widgets and Qt\+Quick. 

It processes some system events to remove the window\textquotesingle{}s default title bar, and provides some shared methods for derived classes to call.

On Windows, \begin{DoxyItemize}
\item {\ttfamily dwm-\/blur}\+: Specify a boolean value to enable or disable dwm blur effect, this attribute is available on Windows 10 or later. \item {\ttfamily dark-\/mode}\+: Specify a boolean value to enable or disable the dark mode, it is enabled by default on Windows 10 if the system borders config is enabled. This attribute is available on Windows 10 or later. \item {\ttfamily acrylic-\/material}\+: Specify a boolean value to enable or disable acrylic material, this attribute is only available on Windows 11. \item {\ttfamily mica\+:} Specify a boolean value to enable or disable mica material, this attribute is only available on Windows 11. \item {\ttfamily mica-\/alt}\+: Specify a boolean value to enable or disable mica-\/alt material, this attribute is only available on Windows 11. \item {\ttfamily extra-\/margins}\+: Specify a margin value to change the {\ttfamily dwm} extended area geometry, you shouldn\textquotesingle{}t change this attribute because it may break the internal state. \item {\ttfamily border-\/thickness}\+: Returns the system border thickness. (Readonly) \item {\ttfamily title-\/bar-\/height}\+: Returns the system title bar height, some system features may be related to this property so that it is recommended to set the custom title bar height to this value. (Readonly)\end{DoxyItemize}
