Detailed Description

WindowAgentBase is the base class of the specifiy window agent for QtWidgets and QtQuick.

It processes some system events to remove the window's default title bar, and provides some shared methods for derived classes to call.

◆ setWindowAttribute()

bool QWK::WindowAgentBase::setWindowAttribute ( const QString & key,
const QVariant & attribute )

Sets the platform-related attribute for the window. Available attributes:

On Windows,

  • dwm-blur: Specify a boolean value to enable or disable dwm blur effect, this attribute is available on Windows 10 or later.
  • dark-mode: Specify a boolean value to enable or disable the dark mode, it is enabled by default on Windows 10 if the system borders config is enabled. This attribute is available on Windows 10 or later.
  • acrylic-material: Specify a boolean value to enable or disable acrylic material, this attribute is only available on Windows 11.
  • mica: Specify a boolean value to enable or disable mica material, this attribute is only available on Windows 11.
  • mica-alt: Specify a boolean value to enable or disable mica-alt material, this attribute is only available on Windows 11.
  • extra-margins: Specify a margin value to change the dwm extended area geometry, you shouldn't change this attribute because it may break the internal state.
  • border-thickness: Returns the system border thickness. (Readonly)
  • title-bar-height: Returns the system title bar height, some system features may be related to this property so that it is recommended to set the custom title bar height to this value. (Readonly)

On macOS,

  • no-system-buttons: Specify a boolean value to set the system buttons' visibility.
  • blur-effect: You can specify a string value, "dark" to enable dark mode, "light" to set enable mode, "none" to disable. You can also specify a boolean value, true to enable current theme mode, false to disable.
  • title-bar-height: Returns the system title bar height, the system button display area will be limited to this height. (Readonly)
